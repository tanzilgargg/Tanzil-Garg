Game Design folder

Game projects built around rules, story, feedback, and assets

This folder contains three game design projects: UJESCA, a hybrid physical-digital game; Anti-Monopoly, a Twine modification that critiques Monopoly through choice and system pressure; and Space Crate, a 3D game asset created in Maya and painted in Substance Painter.

The projects are documented as design systems and production artifacts. The pages identify concept, rules, player feedback, visual evidence, and the craft decisions needed to understand how each project functions.

The folder emphasizes how games communicate through constraints and material choices. Rules, turns, scorekeeping, passages, visual feedback, 3D form, and texture painting are treated as design decisions that shape player behavior and worldbuilding.

Hybrid game system

UJESCA

A hybrid game design project documented through physical components, scorekeeping, rule documentation, sound, and photographed play materials. The project focuses on how players understand what to touch, track, and negotiate.

Critical game mod

Anti-Monopoly: Breaking the System

A branching Twine game that uses player choice, consequence, and interface feedback to critique Monopoly's economic logic. The project turns a familiar board-game system into a critical interactive narrative.
